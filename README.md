# Advanced Excel Business Analytics

## Overview
This repository contains projects completed during the Excel Business Analytics course on Coursera (John Hopkins University).

The exercises demonstrate practical applications of Excel analytical tools:

- Goal Seek
- Data Tables
- Solver

## Skills Demonstrated
- Financial and business modelling
- What-if analysis
- Scenario planning
- Optimization techniques
- Decision support modelling

## Project Files
📄 All_task_Hopkins_Finance_Modelling_VB

## Methods Used

### Goal Seek
Used Goal Seek to determine the input value required to achieve a desired business outcome.

### Data Tables
Used one-variable and two-variable Data Tables to analyse how changes in assumptions affect results.

### Solver
Used Solver to optimise business decisions subject to constraints and objectives.

## Key Learning Outcomes
- Building dynamic business models
- Performing sensitivity analysis
- Optimising business scenarios
- Presenting analytical findings in a structured format
